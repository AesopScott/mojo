Other Repositories

Use these repositories as teaching references for multi-agent orchestration, communication, contracts, state, routing, handoffs, and production-oriented agent system design.

LangGraph Use for: Stateful orchestration, graph-based workflows, routing, supervisor patterns, memory-aware agent systems, durable task flow. Teaching angle: Strongest example for serious orchestration: nodes, edges, state, routing, retries, and human approval gates. LangGraph Supervisor Use for: Supervisor-agent patterns, delegation, centralized routing, controlled communication between specialized agents. Teaching angle: Shows how one coordinator can manage specialized agents without every agent talking to everything. CrewAI Use for: Roles, tasks, crews, agent collaboration, structured multi-agent work. Teaching angle: Maps well to “define the work before you build the agent” with role/task/crew vocabulary. CrewAI Examples Use for: Concrete examples and demos. Teaching angle: Practical workflows when students need examples instead of only abstract framework concepts. OpenAI Swarm Use for: Lightweight handoffs, agent transfer patterns, educational multi-agent examples. Teaching angle: Good for simple handoff mechanics. Frame as educational and lightweight, not the default production recommendation. Microsoft Agent Framework Use for: Enterprise agent architecture, multi-agent workflows, Python/.NET agent development, production-oriented patterns. Teaching angle: Shows where enterprise agent frameworks are heading. Microsoft AutoGen Use for: Historical and conceptual multi-agent patterns, agent conversations, collaborative workflows. Teaching angle: Useful conceptually, while noting Microsoft is moving toward Agent Framework as the newer direction. AWS Multi-Agent Orchestration With LangGraph Use for: Cloud deployment thinking, customer support examples, production architecture, AWS service integration. Teaching angle: Orchestration is infrastructure, services, routing, persistence, and operations, not only prompts. Agency Swarm Use for: Agency-style agent structures, organizational metaphors, specialized agent collaboration. Teaching angle: Useful for discussing whether agents should mirror departments, roles, workflows, or capabilities.
Class Framing
  • Swarm = lightweight handoffs.
  • CrewAI = roles, tasks, and teams.
  • LangGraph = stateful orchestration.
  • Microsoft Agent Framework = enterprise agent direction.
  • AWS guidance = production/cloud architecture.
  • Agency Swarm = organization-style agent design.
Suggested Demo Workflow

Customer request -> Intake Agent -> Router -> Research Agent -> Drafting Agent -> Reviewer Agent -> Human Approval -> Send/Publish

  • Swarm: handoffs.
