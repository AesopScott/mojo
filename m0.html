Project Foundation

M0 starts the project brain with an explicit preflight: intent, operator, notes, sources, memory, RAG, evidence, assumptions, decisions, per-skill run notes, and retrieval-ready context before system shape or agent design begins.

Skill

Template

Primary handoff

M1 System Shape receives the foundation, source inventory, EventStorming Lite notes, and M1 handoff questions.

Reset path

/foundation --wipe previews the foundation scaffold cleanup, then removes confirmed MAPS-created artifacts so the phase can restart cleanly.

Required preflight

M0 now starts with memory discovery: ask whether the project has RAG, where notes live if it does not, how to access each location, and whether there are multiple note or RAG stores.

First question

Ask: "Do you have a RAG system for this project? If not, do you have another place where you store notes?"

Locations

Confirm primary and additional notes, memory, source, and RAG locations, then mark which store is canonical and which are mirrors or derived indexes.

Access

Record whether each location is accessed through the file system, an MCP service, or a REST API before asking identity and project-intent questions.

Shared memory helper

The helper is a script, not a background service. Each skill triggers it after creating its phase artifact.

What it is

maps_memory.py is the shared MAPS runtime helper installed with /foundation.

What triggers it

Every MAPS skill now instructs the agent to call maps_memory.py complete-run after the skill creates its primary output artifact.

Why it exists

It gives each skill one named note in the configured notes and RAG locations, while project-foundation.md remains the control document.

Write target Purpose Example
project-foundation.md Appends the timestamped MAPS Skill Run Log row. /shape | M1 | system-shape.md
<notesRoot>/maps-runs/<skill>.md Maintains one human-readable named note per skill. notes/maps-runs/shape.md
<rag.location>/maps-runs/<skill>.md Mirrors the named skill note into the configured RAG location when one exists. memory/rag/maps-runs/shape.md
.maps/rag-updates.json Records changed notes and marks them as needing reindexing. "needsReindex": true
.maps/foundation-preferences.json Stores the structured notes, sources, memory, and RAG locations selected by M0. "notesRoot": "notes" 
python "$CODEX_HOME/skills/foundation/scripts/maps_memory.py" complete-run \
  --project . \
  --skill /shape \
  --phase M1 \
  --output system-shape.md \
  --summary-file system-shape.md \
  --memory-updates "Track decision captured"
