# Phase Alignment

Use this skill in Phase 0 to make the MAPS project concrete before defining or building an agent.

## Workflow

1. Name the MAPS lifecycle and the APS phase sequence.
2. Separate multi-agent scaffold concerns from one-agent build concerns.
3. Confirm artifact locations.
4. Record the first skill, template, tool, and repo catalogs.

## Output

- `templates/phase-alignment-brief.md` completed for the project
- Initial catalog entries
- Open questions for later phases
