# Scaffold

Use this skill when starting a new MAPS project or rebuilding the MAPS structure in an existing project.

## Inputs

- Project name and repository name
- Lifecycle phase names
- Initial agent or capability focus
- Known tools, repositories, templates, and catalogs

## Workflow

1. Confirm the MAPS and APS boundaries.
2. Create the phase folders, skill folders, template folders, and catalog files.
3. Add the first phase navigation and resource index.
4. Record open questions and follow-up decisions.

## Output

- MAPS scaffold
- Phase resource catalogs
- Phase 0 alignment brief
