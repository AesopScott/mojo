Loop termination

Guidance: Continue only on stop_reason tool_use; terminate on end_turn. Do not parse assistant prose.

Guidance: Emit multiple Task tool calls in one coordinator response, with scoped prompts.

Hook selection

Guidance: Use PostToolUse hook so Claude receives normalized date format.

Tool interception

Guidance: Use deterministic tool-call interception/prerequisite gate; prompt is guidance only.

Session resumption

Guidance: Inform the session of changed files or start fresh with structured summary when old tool results are stale.

Scenario 10
Handoff
