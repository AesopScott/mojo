# Prior-Art Notes

This is a first-pass search summary, not a legal prior-art opinion.

## Search Conclusion

The broad idea of AI-assisted software generation from natural-language requirements is crowded. The stronger patent position is the specific Mojo workflow: two request modes, overview planning, requester approval, provider approval, detailed planning, controlled AI build, validation, and gated deployment.

## Closest References Found

### US20250061402A1

Title: Crowdsourced price based software feature implementation

Link: https://patents.google.com/patent/US20250061402A1/

Why it matters:

- Discusses feature requests.
- Discusses implementing requested features.
- Includes requests that may relate to an existing product or new software product.

Potential distinction:

- Appears focused on price/crowdfunding/ROI-based feature implementation.
- Does not appear centered on the specific two-stage requester/provider approval workflow with AI-generated overview and detailed implementation plans.

Risk level: High.

### US20250165890A1

Title: Automated software development workflows via multi-agent computational framework

Link: https://patents.google.com/patent/US20250165890A1/

Why it matters:

- Covers AI agents for planning, coding, testing, and committing software.
- Includes approval and workflow concepts.

Potential distinction:

- More focused on multi-agent software development mechanics.
- Less focused on customer-facing product/feature request intake and provider-controlled approval lifecycle.

Risk level: High.
